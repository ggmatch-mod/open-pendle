# OpenPendle — `app/`

The OpenPendle frontend: Vite 8 + React 19 + TypeScript + Tailwind v4, wagmi/viem, RainbowKit. Fully client-side and static.

See the [repository README](../README.md) for what OpenPendle is, and [CONTRIBUTING.md](../CONTRIBUTING.md) for how to work on it.

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # tsc -b && vite build → dist/
npm run lint     # oxlint
```

Requires Node 22+ (see `.node-version`).
